# БиоФатихова - Персональный сайт преподавателя биологии

Персональный веб-сайт Маргариты Фатиховой, преподавателя биологии и репетитора.

## Описание проекта

Маркетинговый сайт для русскоязычных родителей и школьников 5-11 классов. Основная конверсия - запись в Telegram на бесплатный 20-минутный пробный урок.

Сайт включает:
- Hero-блок с фотографией преподавателя, CTA и фактами доверия
- Блок о бесплатном пробном уроке
- Раздел "Кому подойдут занятия"
- Примеры занятий
- Стоимость услуг
- Отзывы учеников
- FAQ и контактную информацию

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON для хранения данных (уроки, отзывы)
- Адаптивный дизайн для всех устройств
- Без сборки: GitHub Pages отдает статические файлы напрямую из `main`

## Структура проекта

```

```

## Контент и дизайн

- Текущий сайт является источником правды для формулировок и дизайн-решений.
- Основной CTA: бесплатный 20-минутный пробный урок через Telegram.
- Основные секции в текущей версии: `home`, `diagnostic`, `fit`, `lessons`, `pricing`, `testimonials`, `faq`, `contact`.
- Отдельной секции `about` сейчас нет: образование, опыт, рейтинг Profi.ru и фото преподавателя вынесены в hero-блок.
- Hero использует изображение `images/profile/new_profile.png`.
- Данные галереи и отзывов редактируются в `lessons.json` и `testimonials.json`.
- Дизайн использует теплую бумажную палитру, ботанические акценты, Playfair Display, IBM Plex Sans и IBM Plex Mono.

## Установка и запуск

1. Клонировать репозиторий
2. Запустить любой статический сервер из корня проекта, например:
   ```bash
   python3 -m http.server 8000
   ```
3. Открыть `http://localhost:8000`

Важно: сайт загружает `lessons.json` и `testimonials.json` через `fetch`, поэтому прямое открытие `index.html` через `file://` может блокировать данные в браузере. На GitHub Pages и любом обычном статическом хостинге это работает без дополнительной сборки.
